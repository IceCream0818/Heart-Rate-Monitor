# Heart Rate Monitor
This is an Arduino based heart rate monitor device that detects heart rate(BPM) using the MAX30105/MAX30102 sensors and displays the results on an 8x2 LCD screen.

# How to Use
1. Wire the components according to the KiCad schematic provided in this repository.
2. Download all files from the src folder, open 2x8_display.ino in the Arduino IDE and upload it to your board.
3. Place your finger on the sensor. The LCD will slowly update with your average BPM.
